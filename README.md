## Pré-requisitos 
- Angular
- Node.js
- Java
- IDE

## Comandos
Ver a versão do Node.js:
>node -v

Realizar o build:
>npm install -g @angular/cli

Ver a versão do Angular:
>ng v

Criar um projeto:
>ng new spaangular --prefix=spa <br>
Would you like to add Angular Routing? Yes <br>
Which stylesheet format would you like to use? CSS

Iniciar o servidor:
>ng serve -o
